⚙️ Connect GitHub — One-time Setup
Important: Without this step nothing actually saves. Your changes will appear to work but team members and admin will NOT see each other's updates.
If you are a team member: Ask Nilesh for the dashboard token (one token works for everyone). Paste it below — that's it. Skip steps 1 & 2.
If you are Nilesh (admin) setting this up the first time: follow steps 1 → 2 → 3 below to create the token, then share it with your team.
1
Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token (classic). IMPORTANT: set Expiration to "No expiration" so it doesn't stop working in 90 days.
2
Give it a name like mmm-dashboard → tick the repo checkbox (full repo access) → click Generate token → copy the token immediately. Save it somewhere safe — GitHub will not show it again.
3
Paste the token below (starts with ghp_) and click Save. The app will check it with GitHub before saving — you'll see ✓ if it works.
Your token is stored only in this browser on this device. If you switch devices or use incognito mode, you'll need to paste it again.
